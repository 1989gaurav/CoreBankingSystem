Overview  

Core Banking Solutions is now an emerging need of almost all the Banking Organisations of the entire world. In the years to come, not only Core Banking but also provision of Internet Banking or the e-Banking facilities shall become mandatory. Here we propose a fine look to a common notion of the merging of Core Banking and e-Banking Softwares.

Since this application has both the features of Core Banking and e-Banking, we thought of perspecting this through 3 angles. But when system security comes into play, the vantage points become 4. We have divided our users into 4 categories namely- Visitors, Customers, Employees and Administrator. Each category occuring earlier in the order has functions which are subsets of functions of category occuring later. Hence, the administrator is the Supreme Authority.

As you will move along this manual, you will come across the other features we have implemented in this project. We have implemented certain new concepts such as Issuing of Standing Instructions, Bill & Tax Payments, Remittances, etc. Besides using the advances of technology, we have tried to implement as many security features as possible so that data intrusion becomes inevitable.

Hope that this work of ours stands to the expectation of the users.
